## How to Use These Examples

Each usage example includes:

1. **Description**: A brief explanation of what the example demonstrates
2. **Code**: Complete, runnable code in multiple programming languages
3. **Output**: Screenshots or videos showing the expected result
4. **Resources**: Download links for any required assets (when applicable)
