:::tip[SplashKit Audio allows you to load and play music and sound effects.]
SplashKit's audio library allows you to easily load and play music and sound effects within your programs. To get started with audio the first thing you need to do is load a sound effect or music file. You can do this by calling the `Load Sound Effect(string Name)` function to the `Load Music(string Name)` function.
:::

import { Tabs, TabItem } from "@astrojs/starlight/components";

## Has Music [ðŸ”—](#has-music)

Determines if SplashKit has a music file loaded for the supplied name. This checks against all music files loaded; those loaded without a name are assigned the filename as a default. If this returns `False`, you may want to use [`Load Music`](/api/audio/#load-music) to load in a specific sound effect and give it the desired name.

**Return Type**

- Boolean

Returns `True` if there is a music file with the given `Name` that has been loaded.

**Parameters**

| Name | String | The name to check if a music file is loaded. |

## Load Music [ðŸ”—](#load-music)

Loads and returns a music value. The supplied `Filename` is used to locate the music file to load. The supplied `Name` indicates the name to use to refer to this Music value. The [`Music`](/components/audio/#music) can then be retrieved by passing this `Name` to the [`Music Named`](/components/audio/#music-named) function.

**Return Type**

- [`Music`](/components/audio/#music)

A new music resource.

**Parameters**

## Types

### Music [ðŸ§¾](#music)

The Music type is used to refer to music that can be played by SplashKit. Music files are loaded with [**Load Music**](/components/audio/#load-music), played using [**Play Music**](/components/audio/#play-music), and must be released using [**Free Music**](/components/audio/#free-music).

Unlike [**Sound Effect**](/components/audio/#sound-effect)s, there can only be one music track playing at a time. Playing a new track will stop the currently playing music and switch to the new track.


### Sound Effect [ðŸ§¾](#sound-effect)

The [`Sound Effect`](/components/audio/#sound-effect) type is used to refer to sound effects that can be played by the SplashKit audio code. Sound effects are:

- loaded with [`Load Sound Effect`](/components/audio/#load-sound-effect),
- played using [`Play Sound Effect`](/components/audio/#play-sound-effect),
- and must be released using `Delete Sound Effect` (to release a specific sound effect) or `Delete All Sound Effects` (to release all loaded sound effects).

SplashKit will mix the audio from multiple sound effects, making it possible to play multiple [`Sound Effect`](/components/audio/#sound-effect)s, or even to play the same [`Sound Effect`](/components/audio/#sound-effect) multiple times.

You can check if a [`Sound Effect`](/components/audio/#sound-effect) is currently playing using [`Sound Effect Playing`](/components/audio/#sound-effect-playing).

To stop a [`Sound Effect`](/components/audio/#sound-effect) playing, use [`Stop Sound Effect`](/components/types/#stop-sound-effect). This will stop all instances of this one sound effect from playing.

Use [`Music`](/components/audio/#music) for background music for your games.
