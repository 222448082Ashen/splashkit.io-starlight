:::tip[Terminal]
SplashKit Terminal allows you to read and write values to the terminal in a consistent manner.

## Activate Advanced Terminal [ðŸ”—](#activate-advanced-terminal)

Start using the advanced terminal. Once you call this you will need
to make sure you call [`Refresh Terminal`](/components/terminal/#refresh-terminal) to show anything you have
written. This will allow use of colors, bold, positioning, and other
advanced options.

## Move Cursor To [ðŸ”—](#move-cursor-to)

In advanced mode, this will move the cursor to a given col, row of the
terminal. You can check the terminal size using [`Terminal Width`](/components/terminal/#terminal-width) and
[`Terminal Height`](/components/terminal/#terminal-height). If you try to move outside this bounds then then move
will not occur.

**Return Type**

- `Void` 

**Parameters**

| Name   | Type                                               | Description                                                                        |
| ------ | -------------------------------------------------- | ---------------------------------------------------------------------------------- |
| x | `Integer` | The column to move to, must be between 0 and `terminal_width` |
| y | `Integer` | The row to move to, must be between 0 and `terminal_height` |

**Signatures**

<Tabs>
  <TabItem label="C++">
    <Code code={`void move_cursor_to(int x, int y)`} lang="cpp" />
  </TabItem>

## Set Terminal Colors [ðŸ”—](#set-terminal-colors)

In advanced mode this allows you to set the color of the foreground and
the background. The foreground is the color of the text.


Note that only the following colors are guaranteed to work on all
Terminals (others may work):


- [`Color Black`](/components/color/#color-black) 

- [`Color Dark Gray`](/components/color/#color-dark-gray) 

- [`Color Gray`](/components/color/#color-gray) 

- [`Color White`](/components/color/#color-white) 

- [`Color Red`](/components/color/#color-red) 

- [`Color Dark Red`](/components/color/#color-dark-red) 

- [`Color Green`](/components/color/#color-green) 

- [`Color Dark Green`](/components/color/#color-dark-green) 

- [`Color Blue`](/components/color/#color-blue) 

- [`Color Dark Blue`](/components/color/#color-dark-blue) 

- [`Color Cyan`](/components/color/#color-cyan) 

- [`Color Dark Cyan`](/components/color/#color-dark-cyan) 

- [`Color Light Yellow`](/components/color/#color-light-yellow) 

- [`Color Yellow`](/components/color/#color-yellow) 

- [`Color Magenta`](/components/color/#color-magenta)

- [`Color Dark Magenta`](/components/color/#color-dark-magenta) 


**Return Type**

- `Void` 

**Parameters**

| Name   | Type                                               | Description                                                                        |
| ------ | -------------------------------------------------- | ---------------------------------------------------------------------------------- |
| foreground | [Color](/components/types/#color) | The color of text that is drawn. |
| background | [Color](/components/types/#color) | The color of the background behind drawn text. |

**Signatures**
