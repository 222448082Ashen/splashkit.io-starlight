## Open Window [ðŸ”—](#open-window)

Opens a new graphics window. The supplied `caption` is used to
name and identify the window. The supplied `width` and `height` indicate
the size of the window. The window can be retrieved by passing `caption`
to the `window_named` function.

**Return Type**

- `window`

**Parameters**

| Name   | Type                                               | Description                                                                        |
| ------ | -------------------------------------------------- | ---------------------------------------------------------------------------------- |
| caption | `String` | The title of the window. If a window with this caption
already exists, SplashKit will alter the caption to
ensure that it is unique. |
| width | `Integer` | The width of the window |
| height | `Integer` | The height of the window |

### Window [ðŸ”—](#window)
The Window type is used to refer to a window that you have opened in SplashKit. You create new Windows using [Open Window](/components/windows/#open-window) and they can be closed using [Close Window](/components/windows/#close-window) .
- opened with [Open Window](/components/windows/#open-window) 
- and must be released using [Close Window](/components/windows/#close-window) (to close a specific window) or `Delete All Windows` (to close all loaded windows).
